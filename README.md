 - Author: Daniel J. Umpierez
 - License: MIT
 - Version: 0.1.0

## Description

Logging4Humans is a Python 3 application logging library designed
for humans beans.

## Usage

```py
>>> from logging4humans import Logger
>>> log = Logger('app', 'INFO')
>>> log.info('Starting application ...')
"[Oct-23 23:10:04][app][INFO] Starting application ..."
```
